# Ready, Set, Research! - Syllabus Repository for the Junior Seminar (CMPSC 580)

This repository contains the syllabus for the spring semester course in
research methods in Computer Science, Software Engineering, Data Science, and
Informatics. The purpose of this course is to introduce students to research
methods in computer and information science and to help them to create
a final project report, and its supporting materials, in preparation for
the senior comprehensive project. Specifically, it contains the following files:

- [CMPSC 600 Fall Semester Syllabus](/cmpsc-600-fall-syllabus.md)
- [Metadata Document for Senior Comprehensive Project](/metadata.md)
- [Issue Tracker for Feedback on a Syllabus](https://github.com/Allegheny-Computer-Science-600610-25-26/ready-set-research-syllabus/issues)
- [Pull Requests for Improvement to the Syllabus](https://github.com/Allegheny-Computer-Science-600610-25-26/ready-set-research-syllabus/pulls)
